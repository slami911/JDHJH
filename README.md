## 📦 التثبيت

### المتطلبات الأساسية
- متصفح حديث (Chrome، Firefox، Edge، Safari)
- حساب Firebase (اختياري للمزامنة السحابية)

### التشغيل المباشر
1. افتح الملف `Personal Salary Manager.html` في المتصفح
2. ابدأ بإدخال بياناتك فوراً — كل شيء يعمل محلياً

### تفعيل Firebase (اختياري)
1. اذهب إلى [Firebase Console](https://console.firebase.google.com)
2. أنشئ مشروعاً جديداً
3. فعّل **Authentication** (Email/Password + Google)
4. فعّل **Firestore Database**
5. انسخ إعدادات Firebase إلى الملف:
   - ابحث عن `FIREBASE_CONFIG` في الكود
   - استبدل `YOUR_API_KEY`، `YOUR_PROJECT_ID`، إلخ بمشروعك

## 🎯 كيفية الاستخدام

1. **الصفحة الرئيسية**: تعرض جميع المؤشرات المالية
2. **المصاريف**: أضف مصاريفك مع الفئة والتاريخ والملاحظات
3. **الفئات**: خصص فئات بأيقونات وألوان تناسبك
4. **التقارير**: اعرض وصدر تقارير متنوعة
5. **الإعدادات**: ضبط الراتب، العملة، اللغة، والسمة

### اختصارات لوحة المفاتيح
| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl+1` | لوحة المعلومات |
| `Ctrl+2` | المصاريف |
| `Ctrl+3` | الفئات |
| `Ctrl+4` | التقارير |
| `Ctrl+5` | الإعدادات |
| `Ctrl+N` | إضافة مصروف جديد |
| `Ctrl+F` | بحث في المصاريف |
| `Escape` | إغلاق النافذة المنبثقة |
